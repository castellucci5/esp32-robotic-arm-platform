# CAD Files

This directory contains all CAD-related resources for the project.

## Folder Structure

- `3mf/` → Bambu Studio project files
- `step/` → STEP files
- `stl/` → STL files
- `solidworks/` → Native CAD files

## Bambu Studio Files

⚠️ Some parts require a scale factor of **105%** in Bambu Studio.

Only the following files need to be scaled to 105%:

- `braccio1.3mf`
- `braccio2.3mf`
- `cover_braccio_1.3mf`
- `pinza_destra_e_sinistra_uguali.3mf`
- `ingranaggio_pinza.3mf`
- `vite_di_fissaggio_giunto_2.3mf`
- `vite_giunto_1.3mf`

All remaining files should be printed at 100% scale.
